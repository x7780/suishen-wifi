# 随身 WiFi 设备使用指南

这个仓库用于存放随身 WiFi 设备的使用说明页面内容，通过远程更新机制，让已发出的设备无需重新刷写文件即可同步最新说明。

## 工作原理

设备内置一个本地 HTML 页面（`openwt_index.html`），用户打开后会自动尝试从本仓库拉取 `content.json`，获取最新的注意事项、导航链接和相关文章。

- 联网时：自动拉取最新内容并缓存到本地
- 断网时：显示上次缓存的内容，从未联过网则显示页面内置的默认内容

## 更新内容

只需修改 `content.json` 并推送到 `main` 分支，用户下次联网打开页面即可看到更新。

## 文件说明

| 文件 | 说明 |
|------|------|
| `content.json` | 页面动态内容，包含导航、注意事项、相关文章 |
| `openwt_index.html` | 设备本地页面文件 |

## 内容结构

`content.json` 包含以下字段：

- `nav` — 顶部导航链接
- `header` — 页面标题和副标题
- `steps` — 注意事项步骤列表
- `articles` — 相关文章链接
